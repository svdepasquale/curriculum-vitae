# Curriculum Vitae — Silvio Valerio De Pasquale

![Build CV](https://github.com/svdepasquale/curriculum-vitae/actions/workflows/build.yml/badge.svg)

Senior Engineering Manager — Cloud, Data & Platform.

**Latest PDF:** [Download](https://github.com/svdepasquale/curriculum-vitae/releases/latest/download/cv.pdf)

Source is LaTeX (`cv.tex`). Every push to `main` triggers a GitHub Actions workflow that
compiles the PDF with [Tectonic](https://tectonic-typesetting.github.io/) and publishes it
to the `latest` release — so the link above is always up to date.

## Build locally

```bash
brew install tectonic
tectonic cv.tex
```

Produces `cv.pdf` in the same directory.
